핵심 게임 방식 선택

우리가 만들 게임의 기본 메카닉 — Picross 3D 원작 스타일 vs 고전 노노그램 스타일

A

Picross 3D 스타일 (닌텐도 원작)

꽉 찬 블록에서 시작 → 불필요한 블록 제거 → 3D 오브젝트(토끼, 자동차 등) 조각

장점

  • 직관적 조작감 (깎기)
  • 결과물이 3D 오브젝트 → 시각적 보상 명확
  • 레퍼런스(닌텐도)가 검증된 재미

고려사항

  • 퍼즐 제작 = 3D 오브젝트 디자인 필요
  • 색상(파랑/주황) 시스템 구현 복잡도↑
규칙: 줄의 숫자 = 그 줄에 남을 블록 수
원형 숫자 = 2개 이상 그룹 / 사각형 = 3개 이상 그룹
B

고전 3D 노노그램 스타일

빈 NxNxN 격자에서 시작 → 숫자 단서 보고 채워야 할 셀 추론·마킹

장점

  • 퍼즐 생성 알고리즘화 쉬움
  • 2D 노노그램 로직 직접 확장
  • 렌더러 간단 (빈 격자에 마킹)

고려사항

  • 결과물이 뭔지 알기 어려움 (추상적)
  • 닌텐도 원작과 다른 경험
규칙: 각 행/열/기둥 단서 = 연속 채움 구간 길이
예: [2,1] = ██·█ (2칸 연속, 1칸 연속)
