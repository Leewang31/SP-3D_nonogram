터치 조작 모델

3D 블록을 모바일에서 어떻게 선택하고 제거하는가

A

레이어 슬라이스 방식

한 번에 한 층(슬라이스)만 보여줌 → 탭으로 블록 제거 → 층 이동 버튼

Layer 2 / 5
◀ 이전 층 다음 층 ▶

장점

  • 탭 정확도 높음 (2D 그리드)
  • 구현 단순
  • 닌텐도 원작 DS 방식과 유사

고려사항

  • 3D 전체 형태 파악 어려움
B

자유 3D 회전 방식

드래그로 카메라 오빗 → 탭으로 보이는 블록 레이캐스트 선택 → 제거

👆 한 손가락 드래그 → 카메라 회전
🤏 핀치 → 줌 인/아웃
👆 탭 → 블록 선택·제거
🔲 길게 누르기 → 플래그 마킹

장점

  • 3D 오브젝트 전체 형태 파악 쉬움
  • 시각적으로 임팩트 있음

고려사항

  • 안쪽 블록 선택 어려움
  • 레이캐스트 구현 복잡
C

하이브리드 (닌텐도 Round 2 모바일 방식)

3D 뷰 유지 + 축 방향 선택 버튼 → 해당 축 슬라이스 하이라이트 → 탭 제거

🔄 드래그 → 카메라 회전
📐 X/Y/Z 버튼 → 편집 축 선택
👆 탭 → 선택 축의 블록 제거
슬라이더로 층 이동 가능

장점

  • 3D 뷰 + 정밀 선택 둘 다
  • 원작 모바일과 가장 유사

고려사항

  • UI 복잡도↑, 학습 비용
  • 구현 시간↑

💡 추천: A → C 순서. 레이어 슬라이스로 먼저 메카닉 검증 후 하이브리드로 업그레이드.
