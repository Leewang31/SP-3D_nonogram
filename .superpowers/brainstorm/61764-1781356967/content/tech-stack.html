모바일 기술 스택

3D 렌더링 + 터치 조작 + iOS/Android 동시 지원

A

Unity (C#)

3D 게임 개발 업계 표준. 모바일 3D에 가장 검증된 스택.

Unity 2022 LTS + URP (Universal Render Pipeline)

장점

  • 3D 퍼즐 게임 레퍼런스 풍부
  • iOS/Android 빌드 원클릭
  • 터치 입력·카메라 컨트롤 에셋 다수
  • 퍼포먼스 최적화 도구 내장

고려사항

  • C# 학습 필요 (처음이면)
  • 에디터 무거움
B

Godot 4 (GDScript)

동물숨기 프로젝트와 동일 스택. 3D 지원 내장, 모바일 export 가능.

Godot 4.x + GDScript + Mobile export template

장점

  • 기존 프로젝트와 스택 통일 → 컨텍스트 재활용
  • 오픈소스, 무료
  • GDScript 간결함

고려사항

  • Godot 3D는 Unity보다 레퍼런스 적음
  • 모바일 export 세팅 까다로움
C

Flutter + three_dart / Flame3D

Flutter로 UI, three_dart로 WebGL 기반 3D 렌더링.

Flutter 3.x + three_dart (three.js 포팅)

장점

  • Flutter UI 생산성 높음
  • iOS/Android/Web 동시 지원

고려사항

  • three_dart 커뮤니티 작음, 레퍼런스 부족
  • 3D 게임 프레임워크로 미성숙
  • 퍼포먼스 불확실

💡 추천: Unity (A) — 3D 퍼즐 모바일 게임에 가장 검증된 스택. 레퍼런스 많고 모바일 빌드 안정적.
