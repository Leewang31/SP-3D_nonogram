프로토타입 범위

무엇까지 만들 것인가 — 핵심 검증 vs 완성도

A

메카닉 검증 (최소)

퍼즐 1개 고정 → 핵심 게임플레이만 검증

  • ✅ 3D 블록 격자 렌더링
  • ✅ 카메라 오빗 + 핀치 줌
  • ✅ 탭 → 레이캐스트 → 블록 제거
  • ✅ 레이어 슬라이더
  • ✅ 힌트 숫자 표시 (각 축)
  • ✅ 정답 판정
  • ❌ 메인 메뉴 없음
  • ❌ 퍼즐 선택 없음
  • ❌ 플래그 마킹 없음

예상 기간: 1~2주

B

플레이어블 데모

퍼즐 3~5개 + 기본 UI + 플래그 시스템

  • ✅ A의 모든 것
  • ✅ 플래그 마킹 (길게 누르기)
  • ✅ 퍼즐 선택 화면 (3~5개)
  • ✅ 클리어 연출 (파티클/애니메이션)
  • ✅ 실수 페널티 시스템
  • ❌ 메인 메뉴 없음
  • ❌ 앱스토어 배포 수준 아님

예상 기간: 3~4주

C

MVP 앱

메인 메뉴 + 난이도별 퍼즐 팩 + 전체 UI 흐름

  • ✅ B의 모든 것
  • ✅ 메인 메뉴 + 레벨 선택
  • ✅ 난이도 3단계 (입문/중급/고급)
  • ✅ 진행 저장
  • ✅ 배경 음악 + 효과음
  • ✅ Android APK 빌드 가능

예상 기간: 6~8주

💡 추천: A로 시작 → 메카닉 검증 후 B로 확장. 3D 퍼즐 게임은 "실제로 돌려봐야" 재미 알 수 있음.
