축 화살표 기즈모 방식

3D 블록 중심을 가로지르는 화살표 → 드래그로 축 방향 슬라이싱

상태 1: 기본 (depth 전부 0)
→
X
↑
Y
↙
Z
세 화살표 모두 표시
상태 2: X 화살표 드래그 (depth=2)
⋯
→
X=2
Y·Z 화살표 숨김, X만 표시

동작 규칙

초기: X=0, Y=0, Z=0 → 세 화살표 모두 표시
X 화살표 드래그 → X depth 증가 → X>0이면 Y,Z 화살표 hide
X=0 복귀 → Y,Z 화살표 다시 표시
슬라이싱 결과: X depth D → X축 [0..D] 범위만 visible
탭 제거: visible한 블록만 레이캐스트 대상

고려할 점

⚠ 카메라 오빗 드래그 vs 화살표 드래그 충돌
화살표 탭 → 드래그 모드 진입 (별도 핸들러)
빈 공간 드래그 → 카메라 오빗 (기존 동작)

→ 화살표에 Collision 영역 필요 (StaticBody3D or Control Area)
✅ 장점: 슬라이더 UI 불필요. 3D 공간 안에서 직관적 조작.
축 전환 버튼도 불필요 → UI 깔끔.
한 번에 한 축만 제어 → 혼란 없음.
